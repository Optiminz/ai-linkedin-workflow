# Voice Samples

Writing examples that show the AI what "good" looks like for you.

## How It Works
The system reads these files to understand your natural patterns —
sentence structure, word choice, tone, how you open and close posts.

## The examples/ Folder
These are real published posts showing three different styles.
Use them as reference for what voice-consistent output looks like.

## Adding Your Own
During first run, the system will ask for your writing samples and
save them here. You can also manually add files anytime:
1. Copy a LinkedIn post you've written that sounds like you
2. Save as voice-samples/your-name-topic.md
3. The system will read it next session

More samples = better voice matching. 3-5 is a good start.
