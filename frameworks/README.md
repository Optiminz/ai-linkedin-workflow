# Post Frameworks

Proven structures for LinkedIn posts. The /lidev skill reads this
folder and suggests frameworks that fit your topic.

## What's Here
- contrarian-authority.md — challenge a common belief
- intention-obstacle-stakes.md — tell a story with stakes
- instead-pattern.md — reframe "stop X, start Y"

## Adding Your Own
Found a post structure you love? Break it down:
1. What's the opening pattern?
2. What's the middle structure?
3. How does it close?
4. Save as frameworks/your-framework-name.md

The system will start suggesting it in future /lidev sessions.
