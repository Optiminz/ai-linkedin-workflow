# Guide 01: Your First Post

**Time:** ~30 minutes
**What you'll have at the end:** A publishable LinkedIn post

---

## Two paths to your first post

### Path A: Claude Code

1. Clone the repo and open the directory in your terminal
2. Run `claude` to start a Claude Code session
3. Say: "I want to write a LinkedIn post about [your topic]"
4. The system will ask for a writing sample — paste a few sentences you've written anywhere (email, doc, previous post, doesn't matter)
5. It'll generate three hook options, walk you through refinement, and produce a ready-to-publish post

### Path B: Any AI tool

1. Clone the repo
2. Open `prompts/lidev-standalone.md`
3. Paste your writing samples where indicated (there's a marked placeholder)
4. Add your topic where indicated
5. Copy the entire prompt into ChatGPT, Claude, Gemini, or wherever you work
6. Follow the guided flow — the prompt handles the structure

---

## Why this works

Most people give an AI a topic and ask it to write a LinkedIn post. The result sounds like every other AI-generated LinkedIn post — opener starting with "In today's fast-paced world," a bulleted list of generic insights, a closing question nobody wants to answer.

This system works differently. Instead of asking one AI to do everything, it gives your AI tool:

- **Voice rules** — what intensity, what tone, what's off-limits
- **Proven structures** — hooks that create tension, frameworks that organize thinking
- **Your actual writing patterns** — pulled from samples you provide

The result sounds like you wrote it — because you guided every decision. The AI fills in the structure; you supply the substance and direction.

---

## Before you publish

Read the post aloud. If you'd never say a phrase out loud, cut it. If a sentence feels like something a generic "thought leader" would write, rewrite it.

Common signs something's off:
- Starts with "In today's..." or "As we navigate..."
- Uses "leverage," "unlock," "delve," "seamless," or "game-changing"
- Lists three things when one would do
- Ends with "What do you think?" when you don't actually want to know
